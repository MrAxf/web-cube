You can install **Web Cube** using multiple options.

## Using NPM

You can install **Web Cube** using **NPM** package manager.

    headings={[

        { key: "deno", title: "deno", icon: "i-simple-icons-deno" },
        { key: "bun", title: "bun", icon: "i-simple-icons-bun" },
]}>
    <TabItem tabKey="npm" containerClass="py-4">
        ```bash
        npm install @web-cube/web-cube
        ```
    </TabItem>
    <TabItem tabKey="yarn" containerClass="py-4" hidden>
        ```bash
        yarn add @web-cube/web-cube
        ```

    <TabItem tabKey="pnpm" containerClass="py-4" hidden>

    </TabItem>
    <TabItem tabKey="deno" containerClass="py-4" hidden>

## Using CDN

You can also use **Web Cube** using **CDN** directly in your HTML file.

```html title="index.html"
<script type="module">
    import { define } from "https://esm.sh/@web-cube/web-cube"
    ...
</script>
```
