## Embed in your page

You can embed **Web Cube** in any web app once the custom element is defined.

```html title="index.html"
<web-cube></web-cube>
<script type="module">
  import { define } from "https://esm.sh/@web-cube/web-cube";
  define();
</script>
<style>
  web-cube {
    display: block;
    width: 100%;
    height: 100vh;
  }
</style>
```

<div class="my-8 overflow-hidden rounded-lg bg-slate-300 dark:bg-slate-700">
  <WebCube />
</div>

## Change the size

You can change the cube dimension by setting the `size` attribute.

```html title="index.html"
<web-cube size="2"></web-cube>
```

<div class="my-8 overflow-hidden rounded-lg bg-slate-300 dark:bg-slate-700">
  <WebCube size={2} />
</div>

## Control the cube with JavaScript

Use the `WebCube` methods to rotate the cube and inspect the current state.

```typescript title="index.ts"
import { WebCube } from "@web-cube/web-cube";

const cube = document.querySelector("web-cube") as WebCube;

await cube.rotateCube({ axis: "x", angle: 90 });
await cube.rotateLayer({ axis: "z", layer: 0, angle: 180 });

const state = cube.getState();
cube.setState(state);
```

Continue with [Properties](/docs/the-webcube-element/properties) and [Methods](/docs/the-webcube-element/methods).
